War The Card Game!

This game is multiplatform and should be able to run on Mac OS X, Windows, and Linux. Click here to view all releases

Running War! on each Operating System

  • Windows: Download WarCardGame-Windows.zip, double-click it, drag WarCardGame.exe out of the window that opens, then double-click WarCardGame.exe to launch the game. Java JRE 1.7_51 is required. Go to java.com and click where it says "Download" to install Java
  • Mac: Download WarCardGame-Mac.zip, double-click it, then double-click WarCardGame to launch the game. You may place it in your Applications Folder if you desire for it to be on your Launchpad. You may get a message saying it was unable to open the application. This is because Java is not installed. You also might get a message asking if you want to install Apple's version of Java, which is perfectly fine, but installs an outdated version. I recommend actually going to java.com and click where it says download to download the latest version of Java.
  • Linux: Find out how to install the latest version of Java on your version of Linux. Download WarCardGame-Other.zip, extract it into your downloads folder, or wherever you want it, then in your terminal type java -jar , then drag the extracted .jar file into the terminal. Press enter, and you should have just launched the game.
